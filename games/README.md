# school-corridor-game

像素风格网页小游戏

## 运行游戏

直接在浏览器中打开 `index.html` 即可游玩。

或者使用本地服务器：

```bash
python -m http.server 8000
```

然后访问 http://localhost:8000

## 自定义

- 编辑 `game.js` 修改游戏逻辑
- 编辑 `index.html` 修改界面样式
- 在 `CONFIG` 对象中调整游戏参数

## 控制

- 方向键或 WASD 移动角色
- 发现异常时回头离开

祝玩得开心！
